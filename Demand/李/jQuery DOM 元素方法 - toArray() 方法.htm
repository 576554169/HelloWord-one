jQuery DOM 元素方法 - toArray() 方法

jQuery DOM 元素方法参考手册

实例

将 li 元素转换为数组，然后输出该数组元素的 innerHTML ：

$("button").click(function(){
  x=$("li").toArray()
  for (i=0;i<x.length;i++)
    {
    alert(x[i].innerHTML);
    }
});

亲自试一试

定义和用法

toArray() 方法以数组的形式返回 jQuery 选择器匹配的元素。

语法

$(selector).toArray()
